## Cross-Phase Integration

### Connected (26/28 wiring points)

All major happy-path flows are wired. 19 Tauri IPC commands consumed by frontend. No orphaned exports.

### Integration Gaps (2)

1. **Tool column removal lacks assignment cleanup** — `remove_project_tool` deletes only the DB tool row. Assignments and filesystem artifacts for that tool persist as hidden orphans. Affects TOOL-03 and indirectly PROJ-03 (misleading counts).

2. **"missing" cell state is dead code** — CSS class `.matrix-cell.missing`, `aggregate_project_sync_status` missing-handler, and AssignmentMatrix rendering all support `missing`, but no backend code path ever produces this status value. Affects SYNC-01.
