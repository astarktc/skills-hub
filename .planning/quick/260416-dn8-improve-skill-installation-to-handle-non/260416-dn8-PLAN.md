```rust
/// Fetch the HEAD commit SHA for a branch without cloning.
/// Uses GitHub API: GET /repos/{owner}/{repo}/commits/{branch}
/// Returns the 40-char hex SHA string.
pub fn fetch_branch_sha(
    owner: &str,
    repo: &str,
    branch: &str,
    token: Option<&str>,
) -> Result<String>
```
- Build URL: `https://api.github.com/repos/{owner}/{repo}/commits/{branch}`
- Send GET with headers: `User-Agent: skills-hub`, `Accept: application/vnd.github.v3+json`
- If token provided, add `Authorization: Bearer {token}`
- Parse response JSON, extract `sha` field (top-level string)
- Use existing `check_github_response` for error handling
- On any error, return the error (caller will handle gracefully)
