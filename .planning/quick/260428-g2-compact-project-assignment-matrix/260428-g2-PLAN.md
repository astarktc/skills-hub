# Quick Task: Compact Project Assignment Matrix

## Task 1: CSS changes for compact table and row delineation

**Files:** `src/App.css`
**Action:** Modify matrix table and row styles

Changes:

1. `.matrix-grid table` — Remove `width: 100%` so the table only grows as wide as its content (skill name column + tool columns). Keep `border-collapse: separate` and `border-spacing`.
2. `.matrix-row` — Add alternating row background using `:nth-child(even)` for zebra striping. Use a subtle background like `var(--bg-element)` at low opacity.
3. `.matrix-row:hover` — Add hover highlight for the row being interacted with.
4. `.matrix-header-row th` — Add bottom border to separate headers from body.
5. `.matrix-skill-cell` — Ensure the skill name column is right-padded enough to stay visually connected to its checkboxes.

**Verify:** Table no longer stretches to full window width; rows have visible alternating stripes; checkboxes are easy to associate with skill names.
**Done:** CSS changes produce compact, readable matrix.
