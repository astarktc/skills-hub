**Commit:** `23c8def`
**Files:** `src/App.tsx`

Replaced the 3-line `handleRefresh` (which only re-read SQLite) with a ~100-line two-phase implementation:

1. **Phase 1 — Update all skills from source repos**: Iterates every managed skill, calls `update_managed_skill` backend command to re-download/re-pull from source. Shows per-skill progress via loading overlay.
2. **Phase 2 — Conditional sync to tools**: Only runs when `autoSyncEnabled` is true. Re-fetches fresh skill list (updated central paths), then syncs each skill to each installed tool directory.
3. **Error handling**: Collects all errors and shows summary toast at end. Continues on individual failures.
